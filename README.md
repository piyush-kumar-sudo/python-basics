Welcome! This repository contains a collection of fundamental Python scripts designed to demonstrate 
core programming concepts. Whether you're refreshing your memory or just getting started, these examples 
cover the essential building blocks of the language.

This repository includes code examples for:
Variables & Data Types: Understanding strings, integers, floats, and booleans.
Control Flow: Implementation of if-else statements and logical operators.
Loops: Practical use of for and while loops.
Functions: Defining reusable blocks of code with parameters and return values.
Data Structures: Basic operations with Lists, Tuples, and Dictionaries.
